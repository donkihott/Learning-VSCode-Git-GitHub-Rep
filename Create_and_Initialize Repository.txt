GitHub
- create a new repository
Teminal:
git add . - all files of folder
git commit -m "first commit"
git branch -M main # change Name of branch from 'master' to 'main'
git remote add origin https://(adress of GitHub repository)
git push -u origin main




or:
- Sourse Control (Ctrl+Shift+G) button: 'Publish to GitHub'
